# Question Bank v1 (Thinking Partner)

This is the source of truth for AI-selected follow-up questions.
All questions are manually written and static.
The system must select exactly one question per thought log.

Constraints:
- 5-7 questions per thinking mode
- No advice
- No reassurance
- No emotional validation
- No interpretation

## Rumination (6)
1. What new fact is present now that was absent in the previous entry?
2. Which part of this thought has repeated without change?
3. What unresolved decision is embedded in this loop?
4. What specific uncertainty is still open?
5. What detail in this thought changed since the last time?
6. What single statement keeps recurring here?

## Planning (6)
1. What is the next concrete action?
2. What must happen before that action starts?
3. What is the required sequence after the next action?
4. What deadline or time boundary applies?
5. What dependency is currently missing?
6. What is the smallest executable step?

## Emotional Processing (6)
1. What emotion label is most accurate right now?
2. What is the current intensity from 0 to 10?
3. Where is this emotion most noticeable in the body?
4. Is the intensity rising, stable, or falling right now?
5. What thought is present at the same time as this emotion?
6. What changed in this emotion since the previous check-in?

## Theory / Abstraction (6)
1. What general rule are you proposing?
2. What concrete example supports this rule?
3. What example does not fit this rule?
4. What assumption carries most of this model?
5. What prediction follows if this model is true?
6. What boundary condition limits this model?
