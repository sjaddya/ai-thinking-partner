# Thinking Modes

These modes classify how a thought is functioning, not what it is about.
Each thought must be assigned to exactly one mode.
No additional modes are allowed.

## 1. Rumination

Definition:
Repetitive thinking that revisits the same concern without adding new facts, decisions, or actions.

Non-definition:
Not planning, not emotional processing, and not theory-building.
If a thought produces a concrete next step or a new model, it is not rumination.

## 2. Planning

Definition:
Thinking aimed at deciding actions, sequencing steps, or preparing execution.

Non-definition:
Not replaying the past, not emotional description, and not abstract model-building.
If no concrete action or sequence can be named, it is not planning.

## 3. Emotional Processing

Definition:
Thinking focused on naming, describing, or observing current emotions and internal reactions.

Non-definition:
Not problem-solving, not decision-making, and not abstract explanation.
If the thought is primarily trying to optimize outcomes, it is not emotional processing.

## 4. Theory / Abstraction

Definition:
Thinking that builds general rules, frameworks, or explanatory models about self, relationships, or the world.

Non-definition:
Not immediate action planning, not direct emotion labeling, and not repetitive looping.
If the thought seeks an immediate concrete action, it is not theory / abstraction.

## Deterministic Tie-Break Rules
Use primary function, then apply this order when overlap appears:
1. Planning
2. Emotional Processing
3. Rumination
4. Theory / Abstraction

If a thought has mixed content, assign the earliest matching mode in this order.
This enforces exactly one mode per thought.
